# OS Banker in Java
## Built by Vikas Nair, for Operating Systems (Lab 3)
## 25 Mar 2018

A Java implementation of two banking algorithms, one optimistic (FIFO) the other pessimistic (Dijkstra).

### Compile
```
javac *.java
```

### Run
```
java Main input/input-?.txt --verbose (<- optional flag)
```
